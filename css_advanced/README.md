# ALX CSS Advanced Project

## Overview
This project is part of the ALX Software Engineering curriculum, focusing on advanced CSS concepts and responsive web design. The goal is to recreate a webpage design from a Figma specification with pixel-perfect accuracy.

## Project Requirements
- Recreate the webpage design provided in Figma
- Implement responsive design for various screen sizes
- Use proper CSS methodologies and best practices
- Ensure cross-browser compatibility
- Follow semantic HTML structure

## Design Specifications
- **Design Source**: [Figma Design File](https://www.figma.com/file/your-figma-link)
- **Fonts**: 
  - Primary: Source Sans Pro
  - Secondary: Spin Cycle OT
- **Colors**: Refer to Figma design for exact color values
- **Breakpoints**:
  - Mobile: < 768px
  - Tablet: 768px - 1024px
  - Desktop: > 1024px
